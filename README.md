# 貪食蛇

## 遊戲介紹

### 1. **遊戲目標:**
   - 吃地圖內隨機出現的水果，吃到加十分。
   - 目標是在不碰到自己的尾巴的情況下，得到最多分數，讓尾巴變得最長///。

### 2. **遊戲步驟:**
   - 玩家須透過wasd鍵控制蛇的方向去吃水果，
   - 就醬。

## 程式結構圖


1. **Setup() - 副函式**
   - 初始化文字介面

2. **Draw() - 副函式**
   - 繪製地圖範圍
   - 繪製蛇的頭尾及水果的初始位置

3. **Input() - 副函式**
   - 偵測玩家輸入，變更蛇頭的方向

4. **Logic() - 副函式**
   - 算尾巴長在哪裡
   - 讓蛇移動
   - 計算加分

5. **main() - 主程式**
   - 讓程式運作
   - 在檔案內紀錄最高得分
6.登入登出系統
